Printzware - Modal

A combination of Bootstrap and Jquery is used to build the Modal Window. Each call to open a modal window must pass a unique identifier. If the same identfiier is used twice a warning will be displayed.

Feedback from Modal:

Open a specific Category

A specific Card Category can be open initially in the PW Modal by passing a catID in the PHP printzware_open_url().

Card Completed?

The PW Modal window returns JSON returnData.addCard=yes when the user clicks the Add to Cart button.

Thumbnail

A temporary link to the card thumbnail on the PW server is usable for 30mins. You can download the thumbnail to your website using PHP printzware_get_thumb().

Confirm/Cancel the card

Confiming the card will generate the PDF available to download in your Printzware Control Panel. You can specify an OrderID when confirming a card aswell as an order string & barcode to appear on P4.

Enable/Disable Cards

In the PW Control Panel you can completely disable cards if needed. You can also review newly added cards.

Tags

In the PW Control Panel you can create card groups called Tags. If that Tag is specified in the printzware_open_url() PHP call only those cards are displayed.
